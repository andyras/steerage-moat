Simple AJAX Call to an NYT API

When this page loads, it's making an AJAX call using the jQuery $.ajax function to query the NYTimes Article Search API. http://developer.nytimes.com/article_search_v2.json#

Please don't distribute the script with my API key included.

Note the "q" property in the object passed to the function. 

      // Built by LucyBot. www.lucybot.com
      var url = "https://api.nytimes.com/svc/search/v2/articlesearch.json";
      url += '?' + $.param({
        'api-key': "171742565481ab1fc4d4ed5e38f63d1c:2:60955541",
        'q': 'Grace Hopper'
      });
      $.ajax({
        url: url,
        method: 'GET',
      }).done(function(result) {
        console.log(result);
      }).fail(function(err) {
        throw err;
      });

Open the DevTools, then refresh the page; examine the Console and Network tabs
